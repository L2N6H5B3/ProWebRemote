Settings
Settings
Continuous Playlist
Allow continuous scroll through playlist items.
Retrieve Entire Library on Load
If ticked, entire Library will be downloaded upon connection to speed up the experience. If you are having issues with ProPresenter crashing on connection, uncheck this feature.
Force Next/Previous Slide
If a presentation from a different location (Library/Playlist) than the previous presentation is selected remotely, ProPresenter will default to the previous presentation on next/previous slide. Check this feature to force progression using the remotely-selected presentation.
Follow ProPresenter Presentation
By default, presentations will be automatically changed to follow along with ProPresenter. Uncheck this feature to ignore presentation changes in ProPresenter.
Enable Advanced Live Display
By default, the live display only displays a static image representing the presentation. Check this feature to utilise ProPresenter's network Stage Display protocol to show a current stream of the live slide as the live display.
Save Browser Cookies
If ticked, browser cookies will be saved to retain user preferences between sessions. If you do not want to retain user preferences, uncheck this feature.
